```bash
java -jar target/dependency/webapp-runner.jar --port 8081 target/*.war
```

#### 2. **Database Connection Failed**
```
Error: Connection refused / Timeout
```
**Solution:**
- Verify Supabase credentials in `application.properties`
- Check if SSL is required: `?sslmode=require`
- Ensure firewall allows outbound connections
- Test connection in Supabase dashboard

#### 3. **Maven Build Failure**
```
Error: Cannot resolve dependencies
```
**Solution:**
```bash
# Clean Maven cache
mvn clean

# Force update dependencies
mvn clean install -U

# Skip tests if needed
mvn clean package -DskipTests
```

#### 4. **Java Version Mismatch**
```
Error: class file has wrong version
```
**Solution:**
```bash
# Check Java version
java -version

# Should show Java 21

# Set JAVA_HOME if needed
export JAVA_HOME=/path/to/jdk-21
```

#### 5. **Login Credentials Not Working**
**Solution:**
- Verify migration script ran successfully
- Check data in Supabase dashboard:
  ```sql
  SELECT * FROM admin WHERE mailid='admin@demo.com';
  SELECT * FROM customer WHERE mailid='shashi@demo.com';
  ```
- Ensure passwords match: `admin` and `shashi`
